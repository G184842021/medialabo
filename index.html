「Webコンテンツ作成」の成果

本テーマの学習目標

Webコンテンツ作成の授業を通してWwbの構造、Webページの見た目の設定方法、インタラクティブな処理を記述する基本的な方法がわかるようになる。HTML, CSS, JavaScript の構文リファレンスから，構文の書式や使用方法の情報を入手できる、画像・音声・動画ファイルを組み合わせて簡単な Web ページが作成できる。

プロジェクト課題について

プロジェクト課題では世界の天気予報のWebページの作成をを実施する。ここでページに表示するデータは最低限に、都市名、天気、最高気温、最低気温の４つにした。都市の天気の検索にはラジオボタンを使用し、ぱっと見でどこの国の都市を検索できるのかわかりやすくまた、ボタンを押すだけなので簡単に検索することができるようにした。次に、このページが天気を検索するWebページとぱっと見で分かりやすいように背景を雲の画像にした。

工夫点・苦心点・強調点・感想

情報メディア実験は前半がロボット制作、後半がWebページ作成でそれぞれ授業時間が少なく少し速足で授業が進み少し大変だと感じた。またHTML、CSS、JavaScriptと触る言語が多く楽しくはあったが6回の授授業では覚えることが多く大変であった、なので私は半年間使ってゆっくり、しかっりとWebページの作成の授業を学んでいきたかったと思った。私は６回の授業の中で第４回のDOMとJavaScriptの回が一番難しいと感じた、この回は授業がかなり速足で進んでおりさらに、やることが多かったのでかなり苦戦した。しかしCSSやJavaScriptはWebページに動きがでて自分でページを作っている感が強く、楽しかった。CSSやJavaScriptは調べるとかなりやり方が出てきて学ぶことがものすごく多いことが実感した。Webページ作成にTMLを使うことは知っていてがそこにJavaScriptやCSSを使うことは知らなくWebページの作成には多くの言語を使い、Webページ作成が大変であることが分かる授業であった。課題の3-1で出た数当てゲームは第３回の授業で基礎、第４回の授業でコンソール表示からHTMLに表示、第５回の授業で回答ボタンの設置などし完成と、毎回の授業で学んだことを使い、授業が進みつれプログラムが組みあがっていくので、新しくできるようになったことが分かりやすく新しいプログラムを使うことが出来きかなり楽しく勉強することができたと私は感じた。次にプロジェクト課題は最初は全くどういう感じにするのかが分からなく全く手に付けられなく大変であった、一番最初に天気かグルメか番組表か決めることになっていたがよくわからないうちに決めることになっていたので最初に完成形を少し見せるなど完成までの道のりをもう少しわかりやすくしてほしいと感じた。また授業時間が少ないので仕方ないのもあるが本格的に組めるようになるのが第5回の授業が終わってからなので時間が少なく少し苦労した、しかしたった６回の授業であのくらいのページを作れるようになるのが凄いと感じたのと、普段何気なく使ってるWebページの作りが少しわかった気がし、また難しいことが分かった。Webページ作成の授業で普段使っているWebページがどのように作られているのか、また、HTML、SCC、JavaScriptの基礎を学ぶことができた。HTMLとCSSはjavaとは全く違いい経験になる授業であった。
